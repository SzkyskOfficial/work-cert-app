# 作業証明書アプリ

`index.html` をスマホまたはPCのブラウザで開いて使用します。

## 使い方

1. 上部フォームに入力（日時、会社名、現場名、作業員、作業内容、備考）
2. 必要に応じて `作業員 +2` `作業内容 +1` `備考 +1` で段数を追加
3. `サイン入力` から確認印を手書き入力して保存
4. `保存` でローカル保存、`読込` で復元
5. `PDF作成` でPDFダウンロード、`PCへ送信（共有）` で共有メニューから送信

## 対応済み仕様

- 日勤 / 夜勤のトグル丸囲み表示
- 年月日入力 + 曜日自動表示
- 会社名 / 現場名 / 作業内容リストの自動保存
- 作業員の所属会社付きマスター管理
- 総工数と早残合計の自動計算
- 請 / 常のトグル丸囲み
- 備考段数追加
- 確認印の全画面手書きパレット + 画像保存
- 固定外枠内での段数増加時の行間・フォント自動調整
